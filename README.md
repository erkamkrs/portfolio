- 📱💻 Responsive
- 🎨 Styled with Flask Bootstrap and Css with easy to customize colors

## Getting Started

Clone down this repository. You will need PyCharm or VS Code, and git installed globally on your machine.

### 🛠 Installation and Setup

1. Installation: 

```
npm install
```

2. In the project directory, you can run: 

```
npm start
```

* Any modifications needed to be made to files/folders

### Instructions

* Run the app in the development mode.
Open http://localhost:5000 to view it in the browser. 
* The nav bar includes all the pages such as: "Portfolio", "Contact", "About Me". 
* In order to return to home page click on "WELCOME" on top left corner of every web page.
* To see my recent projects scroll down on home page and click "Check my recent Projects" or click on "Portfolio" from Navbar.
